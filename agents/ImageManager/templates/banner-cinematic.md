# Template: Banner Cinematográfico

**Uso:** Banners padrão para Twitter (posts técnicos, tutoriais, showcases)

## Prompt Base

```
Professional tech banner with text '[HOOK_TEXT]'. Modern cinematic style inspired by A24 film stills.

## Typography
- Render text using High-End Modern Serif (Didot/Bodoni) with wide letter-spacing (0.15em)
- Bold, impactful, high contrast
- Strategic negative space placement with partial masking

## Composition
- Advanced composition using rule of thirds and asymmetry
- Text positioned in upper or lower third (avoid center)
- Deep layered effect with text partially masked by depth

## Style
- "conceptual fine-art photography" referencing A24 film stills
- Chiaroscuro lighting with volumetric Tyndall effect
- Restricted desaturated palette: charcoal background, midnight blue, amber accents

## Technical
- 35mm or 85mm cinematic lens, shallow depth of field (f/1.8 bokeh)
- Non-literal visual metaphor for the theme
- No people staring at camera

## Resolution
- 16:9 landscape (1200x675)

## Output
- Cinematic, high-tech, professional
- Text clearly legible with high contrast
```

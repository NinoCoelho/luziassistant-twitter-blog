# Template: Diagrama Técnico

**Uso:** Diagramas técnicos para explicar conceitos, arquiteturas, fluxos

## Prompt Base

```
Clean technical diagram showing [DESCRIPTION]. Professional technical illustration style.

## Style
- Minimalist white background
- Blue and gray lines, 2-3px thickness
- Clean sans-serif labels (small, legible)
- Clear directional arrows
- Organized, hierarchical structure

## Elements
- Labeled nodes/boxes
- Directional arrows showing flow
- Clear separation between sections
- Consistent spacing

## Composition
- Structured layout (left-to-right or top-to-bottom)
- Grid alignment
- Consistent visual hierarchy

## Resolution
- Variable: 800x600 to 1920x1080

## Output
- Educational, technical, clear
- Easy to understand for beginners
```
